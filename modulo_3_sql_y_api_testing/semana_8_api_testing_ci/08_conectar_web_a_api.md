
## Conectar una Web con una API usando JavaScript

1. **Crear un archivo JavaScript**  
   Crear un archivo `script.js` y enlazarlo desde tu HTML.

2. **Definir las URLs de la API**  
   Establecer la URL base y los endpoints que usarás, como `/users` o `/posts`.

3. **Llamar a los métodos de la API**  
   Usar métodos HTTP como GET para obtener datos y POST para enviar datos, asegurándote de que los nombres de los endpoints y campos coincidan con los de la API.
