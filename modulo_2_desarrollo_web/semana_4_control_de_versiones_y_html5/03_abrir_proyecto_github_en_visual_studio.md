# Apuntes

## Descargar e instalar Visual Studio Code

1. Abre la página oficial:  
   [Descargar Visual Studio Code](https://code.visualstudio.com/download)
2. Descarga la versión correspondiente a tu sistema operativo.
3. Ejecuta el instalador y sigue los pasos de instalación.
4. Una vez finalizada la instalación, abre **Visual Studio Code**.

> Alternativa sin instalación:  
> [Abrir Visual Studio Code Online](https://vscode.dev/)

---

## Descargar e instalar GitHub Desktop

1. Abre la página oficial:  
   [Descargar GitHub Desktop](https://github.com/apps/desktop)
2. Haz clic en **Download GitHub Desktop**.
3. Una vez descargado, haz doble clic en el archivo para iniciar la instalación.
4. GitHub Desktop se abrirá automáticamente al finalizar la instalación.

---

## Clonar (descargar) un proyecto desde GitHub

1. Abre el repositorio que deseas clonar:
   - Pegando la URL del repositorio en el navegador, o
   - Desde la pestaña **Repositories** en GitHub.
2. Encima de la lista de archivos, haz clic en el botón verde **Code**.
3. Selecciona **Open with GitHub Desktop**.
4. Elige la ubicación donde se descargará el proyecto (recomendado: **Escritorio**).
5. Haz clic en **Clone**.

---

## Abrir proyecto en Visual Studio Code

### Opción 1: Desde GitHub Desktop

1. Abre **GitHub Desktop**.
2. Selecciona tu proyecto.
3. Haz clic en **Open with Visual Studio Code**.
4. Se abrirá la carpeta del proyecto en Visual Studio Code.
5. Usa el menú **Explorer** para ver y editar los archivos.

---

### Opción 2: Desde Visual Studio Code

1. Abre **Visual Studio Code**.
2. Ve a **File → Open Folder**.
3. Selecciona la carpeta del proyecto descargado.
4. Haz clic en **Select Folder / Seleccionar carpeta**.
5. Abre el panel **Explorer** para ver los archivos y editarlos.

---

## Commit y Push: subir cambios a GitHub

Después de realizar cambios en el proyecto, debes subirlos a GitHub para que queden guardados en la web.

1. Abre **GitHub Desktop**.
2. Selecciona tu proyecto.
3. Ve a la pestaña **Changes**.
4. Revisa los cambios realizados.
5. Escribe un mensaje descriptivo en el campo de commit.
6. Haz clic en **Commit to main**.
   - El mensaje debe describir claramente los cambios realizados.
   - Si corresponde a una historia de usuario, puedes incluir su enlace.
7. Haz clic en **Push** para subir los cambios a GitHub.
