## Apuntes

El bucle for en Python se usa para iterar sobre una secuencia (como una lista o un rango de números).

### Bucle for
```
for i in range(5):
    print("Número:", i)
```

### Bucle while
```
contador = 0
while contador < 5:
    print("Contador:", contador)
    contador += 1
```
